# Agentic Payments ICP Scorer

A 10x Outbound Engine that finds companies ripe for agentic payments automation.

## How it works

1. **Input your ICP** — describe your ideal customer in plain English
2. **LinkedIn Intelligence** — Apify scrapes LinkedIn for matching companies and people
3. **AI Scoring** — each lead is scored on payment complexity signals
4. **Personalized Outreach** — AI drafts hyper-relevant emails referencing their specific pain

## Tech Stack

- **Frontend:** Next.js + Tailwind CSS
- **Backend:** Node.js API routes
- **Data:** Apify LinkedIn Scraper
- **AI:** OpenAI GPT-4 for scoring + email generation
- **Outbound:** Ready for Instantly/SMTP integration

## Quick Start

```bash
npm install
npm run dev
```

## Environment Variables

```
APIFY_API_KEY=your-apify-key
OPENAI_API_KEY=your-openai-key
```
